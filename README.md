#F1 Simulador de Temporada

##Descripción del proyecto
Este proyecto tiene el objetivo de simular la gestion de una temporada de Formula 1 desarrollado con Python y 
utilizando MySQL como bases de datos.

El objetivo primordial es administrar el ciclo de vida completo de una competición anual, de momento, la funcionalidad definida
del sistema sera la siguiente:

* **Configuracion inicial de la temporada:** Es la definición de la cantidad de circuitos y equipos participantes. 
* **Simulacion rapida de las carreras:** Se trata del procesamiento rápido de los resultados de las carreras.
* **Resultados finales de la temporada:** Debera mostrar los resultados finales de la temporada, equipo ganardor asi tambien
como el piloto ganador.

La Teconologia utilizada sera Python version 3.X, MySQL con un diseño de tablas hasta la 3FN

El inicio de este proyecto sera el modelado de las Bases de Datos
definiendo las entidades asi como sus relaciones.
